# Sailing-optimization
Optimal control project for finding the best path for a boat through a time-dependent vector field.

# Sailing Optimization — Paper + Visualizations + Videos

This repository contains the final paper and supporting video visualizations for the Sailing Optimization project.

- **Repo (source + assets):** https://github.com/ntakashis99/Sailing-optimization

- The **paper** describing the problem setup, model, and results.
- **Video visualizations** demonstrating behavior under different scenarios (motor vs sail, cost settings, etc.).
- A simple **web page** (GitHub Pages) to view everything cleanly on desktop or phone.
